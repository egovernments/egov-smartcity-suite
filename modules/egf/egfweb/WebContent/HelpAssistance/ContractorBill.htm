Contractor Bill

Overview

Screen to enter the contractor bills. Once the bill is passed by the using department, the accounts department will need to enter the bill to recognize the ULB’s liability towards the contractor:

  1. Contractor Bills can be entered only against an existing Work Order. A work order must be created in the system prior to passing a contractor bill.
  2. The cumulative passed amount (for all bills against a specific work order) cannot exceed the Work Order amount.
  3. The account codes that need to be debited must be entered. In addition, certain additional accounts may need to be credited – the account codes need to be entered for them as well. For the standard deductions (eg. TDS, Royalties etc), the deduction types can be selected from a drop-down list (Each deduction type maps to an account code and the % of the passed amount – this is defined in the Masters: Creditor Recoveries). The Contractor Payable Account CANNOT be used on the transaction screen – the amount to this account is automatically calculated.
  4. The Contractor Payable Account is internally assigned. The following rule is applied:

Sum (Debit Accounts) – Sum (Credit Accounts) = Amount credited to the Contractor Payable Account.
